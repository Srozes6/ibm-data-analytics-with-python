# Developer Survey Analysis

## Overview
This project analyzes a developer survey dataset to explore relationships between professional experience, job satisfaction, remote work preferences, educational background, and programming language usage across regions.

## Key Questions Explored
- How does professional coding experience relate to job satisfaction?
- Do remote work preferences vary by employment type?
- Are certain programming languages more popular in specific regions?
- How does educational background relate to employment type?

## Tools Used
- Python
- pandas
- matplotlib
- seaborn

## Outputs
- Cleaned dataset saved for reuse
- Visualizations illustrating key trends
- Clear interpretations included within the notebook
